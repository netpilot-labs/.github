# NetPilot Labs

We build network lab infrastructure for the enterprise.

Modern network teams spend an absurd amount of time on lab setup — provisioning VMs, sourcing licensed device images, wiring topologies, writing baseline configs, and maintaining lab servers that drift out of sync with production. For teams doing continuous change validation, pre-sales POCs, or multi-vendor integration testing, the setup tax isn't an inconvenience. It's the bottleneck.

We're collapsing that work with AI.

**NetPilot** is an AI-powered network lab platform. Network architects describe what they need in plain English — *"Mirror our production BGP setup with the new route policies, validate the failover path, and run iperf across the MPLS backbone"* — and NetPilot generates the topology, vendor-specific configurations, and deploys the entire environment to cloud-hosted Containerlab with real device CLIs. Two minutes, start to validated.

### Built for production teams

- **Change validation** — Mirror your production topology in a sandbox. Test BGP policy changes, ACL updates, and firmware upgrades before they touch live infrastructure.
- **Pre-sales POCs** — Spin up customer demos in minutes, not weeks. Reproducible, shareable, disposable.
- **Multi-vendor integration testing** — Cisco IOL, Juniper cRPD, Arista cEOS, Nokia SR Linux, Palo Alto PAN-OS, Fortinet FortiGate, and FRR running side-by-side in a single topology.
- **Training environments** — Hands-on labs on demand for engineering teams and certification programs, without the lab server overhead.

### Real emulation, not simulation

NetPilot runs actual network operating system instances on isolated cloud VMs. Every device exposes a real CLI — SSH in, run show commands, debug protocols, configure interfaces exactly as you would on production gear. No simplified abstractions, no behavior gaps between lab and reality.
